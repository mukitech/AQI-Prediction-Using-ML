# AQI-Prediction-Using-ML
Air Quality Index (AQI) Prediction is a machine learning project that forecasts air pollution levels based on historical environmental data. The system predicts AQI values using supervised learning models to help assess air quality trends and support early awareness.  This project focuses on building a reliable predictive model using ML

🚀 Project Overview

Air pollution directly impacts public health, climate, and quality of life. AQI is a standardized indicator that converts pollutant concentration levels into a single numerical value representing air quality.

This project:
Trains ML models on historical AQI data
Evaluates performance using regression metrics
Predicts future AQI values based on environmental parameters

📊 Dataset

Preprocessed AQI dataset (cleaned and structured)
Contains pollutant concentration features such as:
PM2.5
PM10
NO₂
SO₂
CO
O₃

Target variable: AQI value

🧠 Machine Learning Approach
1️⃣ Data Preparation

Train-test split
Feature scaling (if required)
Feature-target separation

2️⃣ Models Used

Linear Regression
Decision Tree Regressor
Random Forest Regressor

(Optional) Gradient Boosting / XGBoost

3️⃣ Evaluation Metrics

Mean Absolute Error (MAE)
Mean Squared Error (MSE)
Root Mean Squared Error (RMSE)
R² Score

📈 Results

Compared multiple regression models
Identified best-performing model based on lowest RMSE and highest R²
Demonstrated the ability to predict AQI with strong accuracy on unseen data
